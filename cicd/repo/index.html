Aqualink Automate

APT & DNF package repositories — install and stay up to date with your system package manager.

📖 Read the documentation — install, configuration, HTTP API, MQTT/Home Assistant, Matter, and more.

Debian / Ubuntu / Raspberry Pi OS (APT)

curl -fsSL https://iainchesworth.github.io/aqualink-automate/install-apt.sh | sh

Then sudo apt upgrade keeps it current. Builds for amd64 and arm64 (Raspberry Pi).

Fedora / RHEL / openSUSE (DNF/YUM)

curl -fsSL https://iainchesworth.github.io/aqualink-automate/install-dnf.sh | sh

After installing

sudoedit /etc/aqualink-automate/aqualink-automate.conf   # set serial-port = ...
sudo systemctl start aqualink-automate
journalctl -u aqualink-automate -f
